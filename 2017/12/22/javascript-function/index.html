JavaScript Function

JavaScript中,通常情况下,我们都把代码卸载函数里面,我们大部分代码执行都是函数调用的结果,为了更好的理解Function,我们需要了解javascript对象的一些特性.
javascript对象有以下一些特性:

  • 可以使用字面量创建
  • 可以赋值给变量/数组/其他对象的属性
  • 可以作为函数的参数传递
  • 有属性
  • 可以作为返回值返回

JavaScript函数拥有JavaScript对象的所有特性,而且还可以被调用.那就可以这么说,JavaScript函数就是对象.

函数同样拥有上面的特性:

  • 字面量创建
  • 赋值给变量/数组/和属性
  • 作为参数传递
  • 拥有属性
  • 作为参数返回
